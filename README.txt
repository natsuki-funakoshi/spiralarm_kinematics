Cepheids data are downloaded from https://ogle.astrouw.edu.pl/
->Data Download Site -> ogle4 -> MILKY_WAY_3D_MAP -> Skowron2019_GalClassCephDist.dat

Gaia data are downloaded from https://gea.esac.esa.int/archive/
with the following query:

ruwe < 1.4
non_single_star = 0
radial_velocity_error < 10 (km/s)

and modest limitation close to Cepheids' (RA, Dec).
